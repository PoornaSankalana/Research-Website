Research Project Scope

Have a idea about the scope of our project.

Research Problems


How can those who do not have a basic understanding of telescope behavior and who do not near a telescope observe vivid star patterns?

Generate accurate forecast details using predictive models to give information to the admins about the project to make plans about the future growth as a business.

Developing an efficient and low-latency live-streaming solution that seamlessly integrates with the smart telescope system while ensuring network resilience, user control, and cost optimization?

How can we develop a mobile application with a VR feature that effectively captures head movement using smartphone sensors to provide a seamless and immersive user experience.

Research Objectives


To make a telescope prototype that remotely controls according to real-time data in the mobile application.

Develop predictive model within the admin panel to forecast the recommended number of telescopes that need to supply for a relevant location to minimize the waiting time of users and develop a revenue forecast model to make easy to take decision for the admins about the growth of the project as a business.

Enhance the user experience by implementing live-streaming with video quality adjustment, and payment features to make astronomy accessible and engaging for users.

Develop a mobile app with VR features and tracks head movement using gyroscope and compass sensors, connecting the user interface with the application's logic and backend functions.

Methodology


The Galactic Eye Smart Telescope System includes a mobile app, an admin panel, a telescope, and a server with seamless communication using the Firebase Realtime Database. The users interact with the app to command and receive data from the telescope, while the telescope, in return, sends back observation data to Firebase. A server processes and stores information and real-time updating on Firebase allows users access to recent data. The admin team further observes the system, maintaining data and handling anomalies through Firebase. The predictive models, revenue and telescope supply connected with the admin panel make easier the decision making for future growth of the system as a business. This real-time data interaction method efficiently makes the astronomical results enjoyable and maintains administrative solid control.
